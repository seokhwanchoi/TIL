<h2>1.8 2진 기억 장치와 레지스터</h2>

2진 셀 : 2개의 안정된 상태를 갖고 있다. 정보의 한 비트(1 또는 0)를 기억. 셀에 기억된 정보가
한쪽 안정된 상태에 있으면 1, 다른쪽 안정된 상태에 있으면 0
