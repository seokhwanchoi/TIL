빅오의 정의 : 모든 n, n ≥ n_0에 대해 f(n) ≤ c*g(n)인 두 양의 상수 c와 n_0가 존재한다면 f(n)=O(g(n))이다. (f of n= big-oh of g of n)

주의) f(n)=O(g(n))과 O(g(n))=f(n)은 다르다. '='의 기호는 동등하다의 뜻이아니라 ~이다의 뜻이다.(삽입의 느낌?)

ex)n ≥ 2일 때 3n+2 ≤ 4n이므로 3n+2=O(n)이다.

O(1) O(log n) O(n) O(n log n) O(n^2) O(n^3) O(2^n) 순

ex V1.7)이진 탐색 트리가 균형 잡혀있지 않다고 가정했을 때, 이 트리에서 원소를 찾는데 걸리는 시간(최악의 경우)를 구하라
  A)O(n)--n은 트리에 있는 노드의 개수. 원소 찾는데 걸리는 최대시간은 트리의 깊이와 같다.
  최악의 경우는 트리의 모양이 한쪽으로 길게 늘어진 리스트 형태... 이때 트리의 깊이는 n이 된다.
